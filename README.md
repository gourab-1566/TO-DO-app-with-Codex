
Todo List App
A clean and beginner-friendly Todo List web app built with HTML, CSS, and modern JavaScript. It helps users manage daily tasks with priorities, due dates, subtasks, and local storage support so tasks stay saved in the browser.

✨ Features
Add new tasks
Delete tasks
Mark tasks as completed
Set task priority (low, medium, high)
Add due date and time
Manage subtasks
Save tasks in localStorage
🛠 Tech Stack
HTML
CSS
JavaScript (ES6)

▶️ How to Run
Download or clone the project files.
Make sure all files are in the same project folder.
Open index.html in your browser.
You can simply double-click the index.html file, or run it with a live server if you prefer.
